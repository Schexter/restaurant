## 🛠️ BILD-OPTIMIERUNG WORKFLOW

### 1. Vorbereitung:
```bash
# Originalbilder in hoher Qualität sammeln
# Ordner: images/original/
```

### 2. Bearbeitung:
- Farbkorrektur
- Zuschnitt auf benötigte Größe
- Schärfe anpassen
- Kontrast optimieren

### 3. Optimierung:
```bash
# Tools für Bildoptimierung:
- TinyPNG (Online)
- ImageOptim (Mac)
- RIOT (Windows)
- Squoosh (Google)

# Zielgrößen:
- Hero-Bilder: max. 500KB
- Card-Bilder: max. 200KB
- Thumbnails: max. 50KB
```

### 4. Formate erstellen:
```html
<!-- WebP mit JPG-Fallback -->
<picture>
  <source srcset="image.webp" type="image/webp">
  <source srcset="image.jpg" type="image/jpeg">
  <img src="image.jpg" alt="Beschreibung">
</picture>
```

## 📝 ALT-TEXTE TEMPLATE

### Beispiele für gute Alt-Texte:
```html
<!-- Gut ✅ -->
alt="Eleganter Teller mit Jakobsmuscheln auf Blumenkohlpüree, garniert mit Microgreens"

<!-- Schlecht ❌ -->
alt="Essen"
alt="IMG_1234.jpg"
```

### Alt-Text Checkliste:
- [ ] Beschreibt das Bild präzise
- [ ] Enthält wichtige Details
- [ ] Ist nicht zu lang (max. 125 Zeichen)
- [ ] Vermeidet "Bild von..." oder "Foto von..."
- [ ] Berücksichtigt den Kontext

## 🚀 UPLOAD-ANLEITUNG

### Schritt 1: Bilder vorbereiten
1. Bilder in korrekter Größe exportieren
2. Dateinamen gemäß Liste vergeben
3. Bilder optimieren (Dateigröße reduzieren)

### Schritt 2: In Projekt-Ordner kopieren
```
RestaurantWebsite/
└── images/
    ├── catering/     <- Catering-Bilder hier
    ├── chef/         <- Chefköchin-Bilder hier
    ├── gerichte/     <- Speisekarten-Bilder hier
    ├── rezepte/      <- Rezept-Bilder hier
    └── restaurant/   <- Allgemeine Bilder hier
```

### Schritt 3: HTML aktualisieren
```html
<!-- Beispiel: Bild-Pfad ersetzen -->
<!-- Vorher: -->
<img src="images/catering/gala-dinner.jpg" alt="">

<!-- Nachher: -->
<img src="images/catering/gala-dinner.jpg" 
     alt="Festlich gedeckter Tisch für Gala Dinner mit Kristallgläsern und Kerzen"
     loading="lazy">
```

### Schritt 4: Testen
- [ ] Alle Bilder werden geladen
- [ ] Alt-Texte sind vorhanden
- [ ] Ladezeiten sind akzeptabel
- [ ] Responsive Darstellung funktioniert
